# Media Files for Raiz de Luz

Add your images and media files to this directory.

## Required Images

### Essential:

- `raiz-de-luz-logo.svg` or `.png` - The clinic logo
- `header-image.jpg` - Header/banner image for all pages

### Services Page:

- `massagem-xamanica.jpg` - Shamanic massage image
- `cura-energetica.jpg` - Energy healing image
- `limpeza-fumo.jpg` - Smoke cleansing image
- `produtos.jpg` - Natural products image

### About Page:

- `therapist-1.jpg` - First therapist photo
- `therapist-2.jpg` - Second therapist photo (if applicable)

## Image Recommendations

- **Logo**: SVG preferred, or PNG with transparent background
- **Header**: 1200x400px minimum, landscape orientation
- **Service images**: 600x600px minimum, square or portrait
- **Therapist photos**: 400x400px minimum, professional headshots

## Image Optimization

Before uploading, compress your images:

- Use [TinyPNG](https://tinypng.com/) for PNGs
- Use [Squoosh](https://squoosh.app/) for JPGs
- Keep file sizes under 500KB when possible

## Naming Convention

Use lowercase with hyphens:

- ✅ `shamanic-massage.jpg`
- ❌ `Shamanic Massage.jpg`
- ❌ `shamanic_massage.JPG`
